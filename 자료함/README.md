# 100일 웹앱 자료 넣는 법

## 1. 사진

`사진` 폴더 안에 데이트 날짜별 폴더를 만들어 사진을 넣어주세요.

예시:

```text
사진/
├── 2026-04-04_첫데이트/
│   ├── 01.jpg
│   ├── 02.jpg
│   └── 03.jpg
├── 2026-04-12_한강데이트/
│   ├── 01.jpg
│   └── 02.jpg
└── 2026-05-03_비오던날/
    └── 01.jpg
```

- 폴더명의 날짜는 가능하면 `YYYY-MM-DD` 형식으로 적어주세요.
- 사진 파일명은 정리하지 않아도 됩니다. 촬영 순서도 메타데이터를 확인해 정리할 수 있습니다.
- 대표 사진으로 꼭 쓰고 싶은 파일만 파일명 앞에 `대표_`를 붙여주세요.
- 게임에 넣으면 안 되는 파일은 사진 폴더에 넣지 않거나 `제외_`를 붙여주세요.
- JPG, JPEG, PNG, WEBP가 가장 좋습니다. HEIC도 넣어도 되며 필요하면 변환합니다.

## 2. 영상

짧은 영상은 `영상` 폴더에 넣고, 해당 날짜가 보이도록 이름을 붙여주세요.

```text
영상/2026-04-12_한강_노을.mp4
```

긴 원본 영상도 넣어도 되지만, 실제 웹앱에는 모바일 로딩을 위해 짧게 편집하거나 압축해 사용합니다.

## 3. 데이트 이야기

`데이트-기록.md`를 열고 기억나는 만큼만 작성해주세요. 날짜, 장소, 웃긴 일 한 줄, 그날의 감정 한 줄만 있어도 충분합니다.

## 4. 마지막 편지

`마지막-편지.md`에는 완성된 편지를 쓰지 않아도 됩니다. 꼭 전하고 싶은 말이나 단어만 적어주면 게임 엔딩에 어울리게 초안을 만들어드립니다.

## 개인정보 참고

Render에 공개 배포하면 URL을 아는 사람이 이미지 파일에 접근할 가능성이 있습니다. 외부에 절대 노출되면 안 되는 사진은 넣지 말아주세요. 필요하면 최종 배포 전에 서버 기반 비밀번호 보호를 추가할 수 있습니다.
